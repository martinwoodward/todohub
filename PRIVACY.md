# TodoHub Privacy Policy

**Last Updated: January 2025**

## Overview

TodoHub is a native iOS application that uses GitHub Issues as a todo list backend. This privacy policy explains how TodoHub collects, uses, and protects your information.

## Information We Collect

### Account Information
When you sign in with GitHub, TodoHub accesses the following information from your GitHub account:
- **User ID**: Your unique GitHub user identifier
- **Username**: Your GitHub username (@handle)
- **Display Name**: Your public profile name (if provided)
- **Email Address**: Your GitHub email address (if public or authorized)
- **Avatar URL**: Your GitHub profile picture URL

### Data You Create
- **Todo Items**: The todos you create are stored as GitHub Issues in your selected repository
- **Repository Selection**: Which repository you've chosen for storing todos
- **App Preferences**: Your theme preference and other settings

## How We Use Your Information

All data is used solely to provide the TodoHub functionality:
- **Authentication**: To verify your identity and provide access to your GitHub account
- **Todo Management**: To create, update, and manage issues in your GitHub repository
- **Personalization**: To display your profile information and preferences within the app

## Data Storage

### Local Storage
- **OAuth Tokens**: Stored securely in the iOS Keychain
- **User Preferences**: Stored locally using iOS UserDefaults
- **User Profile**: Cached locally in the iOS Keychain for offline display

### Cloud Storage
- **Todo Data**: All your todo items are stored as GitHub Issues in your own GitHub repository
- **No Third-Party Servers**: TodoHub does not maintain any server infrastructure or database

## Third-Party Services

TodoHub uses the following third-party services:

### GitHub
- TodoHub connects directly to GitHub's API to manage your issues
- GitHub's privacy policy applies to data stored on GitHub: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

### Apple Speech Recognition (Optional)
- If you use voice dictation for creating todos, audio is processed by Apple's Speech Recognition framework
- See Apple's privacy policy for more details: https://www.apple.com/legal/privacy/

## Data Sharing

TodoHub does **not**:
- Sell your data to third parties
- Share your data with advertisers
- Use your data for analytics or tracking
- Store your data on external servers

## Your Rights

You have the right to:
- **Access**: View all your data through your GitHub account
- **Delete**: Remove your data by deleting issues from your GitHub repository
- **Revoke Access**: Remove TodoHub's access to your GitHub account at any time through GitHub Settings > Applications > Authorized OAuth Apps

## Data Retention

- **Local Data**: Deleted when you sign out or uninstall the app
- **GitHub Data**: Retained in your GitHub repository until you delete it

## Security

TodoHub employs industry-standard security measures:
- OAuth 2.0 for secure authentication
- Secure storage of credentials in iOS Keychain
- HTTPS encryption for all API communications
- No storage of passwords or sensitive credentials

## Children's Privacy

TodoHub is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

## Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by updating the "Last Updated" date at the top of this policy.
